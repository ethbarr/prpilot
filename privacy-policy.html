PRPilot – Privacy Policy

Last updated: April 27, 2026  ·  Effective immediately

PRPilot ("the extension", "we", "our") is a Chrome extension that adds AI-powered code review to GitHub pull requests. This policy explains what data the extension collects, how it is used, and what it does not do.

1. Data We Collect and Store

PRPilot stores the following data locally in Chrome's extension storage on your device:

  • Your AI API key (Anthropic or OpenAI) — entered voluntarily by you in the extension popup
  • Your GitHub personal access token (optional) — entered voluntarily if you choose to provide one
  • Extension settings — your chosen AI provider, model, review style, and token limits

This data is stored using Chrome's chrome.storage.local API. It is isolated to the PRPilot extension and is not accessible to websites or other extensions. None of this data is transmitted to PRPilot or any third party other than the AI provider you select.

2. Data Sent to Third-Party Services

When you click "AI Review" on a pull request, the extension:

  • Fetches pull request file diffs from GitHub via the GitHub REST API (api.github.com). If you provided a GitHub token, it is included in the Authorization header of this request. If not, the request is made without authentication (subject to GitHub's public rate limits).
  • Sends the diff content to your chosen AI provider — either Anthropic (api.anthropic.com) or OpenAI (api.openai.com) — using your own API key. The payload consists of the pull request diff and a review prompt. It is subject to the privacy policy of the AI provider you choose.

PRPilot does not have its own servers. No data passes through any PRPilot-owned infrastructure. The only network requests made by the extension are the two described above, directly from your browser.

3. Data We Do Not Collect

  • We do not collect analytics, telemetry, or usage statistics
  • We do not track which repositories or pull requests you review
  • We do not sell, share, or monetize any data
  • We do not use cookies or browser fingerprinting
  • We do not create user accounts or store data on remote servers

4. Third-Party Privacy Policies

When you use PRPilot, your code diffs are sent to the AI provider you configure. Please review their privacy policies:

  • Anthropic Privacy Policy
  • OpenAI Privacy Policy
  • GitHub Privacy Statement

By default, Anthropic does not use API request content to train models. OpenAI's API data usage depends on your account tier and settings. Please verify the current policies with each provider.

5. Permissions

PRPilot requests the following Chrome permissions:

  • storage — to save your API key and settings locally on your device
  • activeTab — to read the current GitHub PR page URL (repository name and PR number)
  • github.com, api.github.com — to fetch pull request diffs via the GitHub API
  • api.anthropic.com — to call Anthropic's API with your API key (if you select Anthropic)
  • api.openai.com — to call OpenAI's API with your API key (if you select OpenAI)

6. Data Deletion

To delete all data stored by PRPilot, click the "Clear Settings" button in the extension popup. This removes your API key, GitHub token, and all preferences from Chrome's local storage. Alternatively, uninstalling the extension will remove all associated local storage data automatically.

7. Children's Privacy

PRPilot is a developer tool intended for adults. We do not knowingly collect any information from children under the age of 13.

8. Changes to This Policy

We may update this policy if the extension's behavior changes materially. The "Last updated" date at the top of this page will reflect any changes. Continued use of the extension after a policy update constitutes acceptance of the revised policy.
