# PRPilot — AI Code Reviewer for GitHub

> Instant, structured AI code review on any GitHub pull request. No copy-pasting. No context switching. Just open a PR and click a button.

![PRPilot in action](docs/screenshot.png)

---

## What it does

PRPilot adds a floating **🤖 AI Review** button to every GitHub pull request page. Click it and within seconds you get:

- **Overall score** (1–10) with colour coding
- **Summary** of what the PR does
- **Flagged issues** — bugs, security flaws, and correctness problems ranked by severity
- **Suggestions** — improvements that aren't strictly wrong but could be better
- **Praise** — things done well (because good code deserves acknowledgement)

Reviews are generated by your choice of **Anthropic Claude** or **OpenAI GPT**, using your own API key. Your key is stored locally in the browser and never transmitted anywhere except the AI provider you configure.

---

## Features

- ✅ Works on any public GitHub PR — no token required
- ✅ Works on private repos with a GitHub personal access token
- ✅ Supports Anthropic (Claude) and OpenAI (GPT-4o, GPT-4o-mini)
- ✅ Four review styles: General, Security, Performance, Concise
- ✅ Handles large PRs gracefully — diffs are truncated intelligently
- ✅ Zero data retention — your code goes to the AI provider and nowhere else
- ✅ Bring your own API key — no PRPilot subscription required to start

## Installation (local / development)

### Prerequisites
- Node.js 18+
- A Chromium-based browser (Chrome, Edge, Brave)

### Build

```bash
git clone https://github.com/ethbarr/prpilot.git
cd prpilot
npm install
npm run build
```

### Load in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode** (toggle, top-right)
3. Click **Load unpacked**
4. Select the `dist/` folder inside the project
5. PRPilot appears in your extensions list

### Configure

Click the PRPilot icon in your Chrome toolbar, then:

1. Choose your AI provider (Anthropic or OpenAI)
2. Paste your API key
3. Select a model and review style
4. Click **Save Settings**

## Usage

Navigate to any GitHub pull request — for example:

```
https://github.com/facebook/react/pull/12345
```

A blue **🤖 AI Review** button will appear in the top-right corner. Click it. The extension fetches the PR diff via the GitHub API, sends it to your configured AI provider, and renders the review in a side panel.

For **private repositories**, add a GitHub personal access token in Settings (the optional "GitHub Token" field). Generate one at [github.com/settings/tokens](https://github.com/settings/tokens) with `repo` scope.

## Privacy

PRPilot sends your PR diff to the AI provider you configure (Anthropic or OpenAI) and nowhere else. Your API key is stored in `chrome.storage.local` — it is local to your browser and is never transmitted to any PRPilot server (there isn't one).

For sensitive or proprietary code, use your own API key under your own account so your provider's data retention policies apply directly to you.
